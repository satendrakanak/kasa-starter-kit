## Avoid For Starter Listing

Do not use screenshots that primarily show disabled starter workflows unless you are preparing a larger edition listing:

- Exams screens.
- Refund screens.
- Tags screens.
- Faculty or live class screens.

## Screenshot Rules

- Use clean demo data only.
- Hide real license keys, emails, secrets, and payment credentials.
- Use current KASA logos before capturing final images.
- Keep browser width consistent across dashboard shots.
- Prefer light mode for marketplace clarity, with one dark-mode preview only if needed.
