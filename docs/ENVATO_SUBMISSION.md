# Envato Submission Notes

## Suggested Item Name

KASA Starter Kit - Next.js NestJS LMS with Docker Installer, License Activation, Courses, Payments, Certificates, CMS

## Short Description

KASA Starter Kit is a production-ready LMS starter built with Next.js, NestJS, PostgreSQL, Redis, and Docker. It includes authentication, admin and student dashboards, course management, paid enrollments, progress tracking, certificates, coupons, basic CMS, and a guided installer with license activation.

## Key Features

- Next.js frontend and NestJS backend.
- Docker-based local and production setup.
- Installation wizard with database setup, academy details, license activation, admin creation, and demo import.
- Admin and Student roles.
- Login, register, email verification, password reset, optional social auth settings.
- Course management with categories, chapters, video lessons, file/PDF attachments, thumbnails, and free/paid toggle.
- Student learning dashboard with continue watching, progress tracking, mark complete, orders, and certificates.
- Completion certificate generation.
- Razorpay-ready checkout and coupon support.
- Admin dashboard with courses, users, orders, coupons, certificates, media, settings, and sales overview.
- CMS pages for articles, testimonials, and contact leads.
- Site settings for branding, SMTP, storage, payments, and social auth.
- KASA light and dark logo assets included.
- REST API with Swagger docs.

## Not Included In Starter Edition

These workflows are intentionally disabled from the starter product surface:

- Faculty workspace.
- Live classes and BBB settings.
- Exams and question bank.
- Push notifications and PWA installation.
- Engagement automation.
- Refund workflows.
- Moderation and advanced review workflows.

## Technical Requirements

- Docker Engine or Docker Desktop.
- Node.js 20+ only if running outside Docker.
- PostgreSQL 16+.
- Redis 7+.
- 4 GB RAM minimum, 8 GB recommended.

## Demo Accounts

If starter demo data is imported, use the demo accounts generated by the seed data or create a fresh admin from the installer. Do not include production credentials in the marketplace ZIP.

- Installer completes with a valid license key.
- Public pages open: Home, Courses, Articles, Testimonials, Contact.

- Disabled routes return not found: Faculty, Exams, Engagement, BBB settings, Push settings.

- Client TypeScript passes.
