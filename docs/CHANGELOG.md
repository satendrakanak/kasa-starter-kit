# Changelog

## 1.0.0 - Starter Kit Release

- Rebranded package as `kasa-starter-kit`.
- Added Docker-first installer flow with database setup, academy details, license activation, admin creation, and demo import.
- Included Admin and Student starter roles.
- Included self-learning course management, video lessons, attachments, progress tracking, and certificates.
- Included payments, coupons, orders, articles, testimonials, contact leads, media, and site settings.
- Added KASA light and dark logo assets.
- Removed PWA and push notification starter surface.
- Removed BBB settings from starter settings.
- Disabled faculty workspace, live classes, exams, engagement, moderation, refunds, and other advanced extension routes from starter navigation.
- Added Envato packaging, installation, screenshot, and release documentation.
